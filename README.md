# 📱 Pokedex

Aplikasi web Pokedex sederhana untuk menjelajahi data Pokémon, menyimpan koleksi pribadi, dan melihat riwayat aktivitas pengguna.

🔗 **Live Demo:** [poke-dex-alpha.vercel.app](https://poke-dex-alpha.vercel.app/)

![Pokemon Logo](https://poke-dex-alpha.vercel.app/tpokemon.jpeg)

## ✨ Fitur

- **Beranda** — Menampilkan data seluruh Pokémon secara lengkap
- **Koleksi Saya** — Kantong koleksi pribadi untuk menyimpan Pokémon favorit
- **Riwayat** — Log aktivitas sistem, mencatat interaksi pengguna di dalam aplikasi

## 🛠️ Tech Stack

- [Next.js]
- [Laravel]
- Vercel 

## 📖 Cara Penggunaan

1. Buka halaman **Beranda** untuk melihat daftar seluruh Pokémon
2. Klik Pokémon yang diinginkan untuk melihat detail
3. Tambahkan Pokémon favorit ke **Koleksi Saya**
4. Cek halaman **Riwayat** untuk melihat log aktivitas yang tercatat
